
# notes-app-react-v2

Simple note apps


### Installation

_Clone down this repository. You will need `node` and `npm` installed globally on your machine._

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run Test Suite 
   ```sh
   npm run dev
   ```
